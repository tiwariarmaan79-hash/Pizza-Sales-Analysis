# 🍕 Pizza Sales Analysis

## 📌 Project Overview

This project analyzes one year of pizza sales data to uncover customer purchasing patterns, sales performance, and business insights. The analysis combines **MySQL** for data querying and **Power BI** for interactive dashboard creation, following an end-to-end data analytics workflow.

The goal of this project is to transform raw transactional data into meaningful insights that can support business decision-making.

---

## 🎯 Business Objectives

* Analyze overall sales performance.
* Identify the busiest days and months.
* Determine top and bottom performing pizzas.
* Understand customer ordering behavior.
* Compare pizza categories and sizes.
* Build an interactive dashboard for business users.

---

## 📈 Key Insights

* Peak order volume occurs on weekends, particularly Friday and Saturday evenings.
* July and January record the highest number of orders.
* Large-sized pizzas contribute the highest share of sales.
* The Classic pizza category generates the highest revenue and order volume.
* Best-selling and least-performing pizzas were identified using revenue, quantity sold, and total orders.
